Python - TkinterTransactions GUI

Transactions Database Management

This is a transaction management application for storing and modifying business transactions. It was built using a series of Python libraries including Tkinter for the GUI, Pillow for image display, and XlsxWriter to convert the data., and SQLite as our RDBMS

Users are able to store information related to their transactions including the Client Name, Date, Payment, and Service Description. This information can then be viewed in a user friendly format, and is easily edited by inputing the corresponding Transaction ID

For users who would like to further adjust and structure their data, or simply prefer a spreadsheet styling, the database can be quickly saved in an xlsx format.
