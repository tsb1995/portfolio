Django - FlaskCalculus App

Taymath

This is an interactive web application created using a Django-Flask framework. Users are provided the tools they need to solve and understand problems related to their introductory calculus courses. Via a minimalist interface, users input the necessary parameters specific to their problem.

To solve these problems in the Python backend, a variety of libraries are implemented. To properly handle the functions given, SymPy first converts all commonly used variable letters from the string into actual variables. Once properly formated, SymPy also handles most of the calculations. Each step of calculations is saved displayed, along with the end result, in a table populated via Javascript and formatted with HTML and CSS

Finally, a group of Numpy arrays are generated for the input space, the function given, as well as whatever feature of the graph that the particular problem requires (in our example, a tangent line representing the approximation function). With this information Matplotlib, along with Seaborn formatting for a nice grid and visual effect, generates a graph which expresses the ideas underlying the solution to the problem

This app is currently fully deployed via Heroku and live at TayMath.com
