PythonZillow Grabber

Housing Data Management

This is an applcation created to take a properly formatted excel housing file, grab relevant housing information from the Zillow API, integrate it into the existing data, clean everything up, and upload to Google Sheets using their API. The GUI was created using Tkinter and data modification and cleaning done with the Pandas library.

Users are able to select which Google account they would like to save their Sheets Document to, and their authentication information will be saved as a pickle file. Their Zillow API Key is stored in the zillowkey.conf config file.
