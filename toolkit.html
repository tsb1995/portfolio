Android - KotlinStudent Management

Tutor Toolkit

This is a fully deployed and functional Android phone application written in Kotlin using the Android Studio SDK. Users are provided the ability to store information relating to their students' grades, history, and personal information locally to their device. They also have access to a task list ehy can edit to fit day to day needs. The information entered by the user is stored in a Relational Database via the Room persistence library which acts as an abstraction layer over SQLite for robust access. With a clean and readable interface, users can quickly input and modify the data they have stored.

This app is currently fully deployed and live on the Google Playstore
