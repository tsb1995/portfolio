TensorFlow.JSSmart Camera

Object Detection in Javascript

This is an implementation of Google's TensorFlow.JS Library. Making use of HTML5, CSS3, and Javascript, this page showcases an application of skills in Web Development and Data Science. Below, you can hold objects up to your webcam to for Real Time Classification! When Ready, click "Enable Webcam" below, and allow access when prompted.

Here you can find an implementation of the facial recognition model BlazeFace. Upon activating your webcam, the model will continuosly scan the video feed and return bounding boces for any faces detected, along with the relative locations and likelihoods of facial landmarks (eyes, nose, mouth, etc).

Below is a TensorflowJS implementation of a custom CNN Model which makes use of two Convolutional Layers, two Max Pool layers, and one Dense Hidden Layer of size 1024, as well as a Dropout Layer to prevent overfitting (with a dropout rate of 20%). The original model was constructed and trained using Tensorflow in Python and a TensorflowJS version of the model was saved. Note that the model was trained on normalized and centered images, and as such, the larger and closer to the center that you draw your numbers, the more accurate the prediction.

The code for this project, along with other applications of Data Science, can be found through my Github account.
